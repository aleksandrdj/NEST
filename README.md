# NEST - Приложение для родителей

Приложение для подбора одежды ребенку по погоде с использованием ИИ.

## Как развернуть на GitHub Pages

Чтобы приложение корректно работало на GitHub Pages, нельзя просто загрузить исходный код. Его нужно **собрать** (build).

Я добавил всё необходимое для автоматизации этого процесса:

1.  **Настройка путей**: В `vite.config.ts` установлен `base: '/NEST/'`. Это гарантирует, что приложение будет искать свои файлы в папке репозитория, а не в корне домена.
2.  **Автоматический деплой**: Я создал файл `.github/workflows/deploy.yml`. 

### Что вам нужно сделать:

1.  **Отправьте (Push) изменения** в ваш репозиторий на GitHub.
2.  Перейдите в настройки вашего репозитория на GitHub: **Settings > Pages**.
3.  В разделе **Build and deployment > Source** выберите **GitHub Actions**.
4.  Теперь при каждом пуше в ветку `main` GitHub будет сам собирать проект и публиковать его.

### Локальная разработка

```bash
npm install
npm run dev
```

### Сборка

```bash
npm run build
```
Результат сборки будет в папке `dist/`. Именно содержимое этой папки должно попадать на хостинг.
